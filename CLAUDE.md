# 法学实证研究项目 v1.1

## 项目说明

本项目是一个**通用的法学实证研究工具包**，用于处理、清洗、整理、结构化大量裁判文书（.doc/.docx），最终输出可直接用于描述性统计和回归分析的结构化数据表格（CSV）。

**v1.1 核心更新**：(1) 数据标准化（比例小数0-1、日期YYYY-MM-DD）；(2) AI幻觉检测与迭代清零程序。

## 核心能力

- **格式转化**：将 Word 裁判文书转为纯文本，剥离页眉页脚
- **数据清洗**：去重、完整性检查、关联案件归总
- **信息抽取**：利用 Agent 集群从文书中提取结构化字段
- **标签统一**：开放文本字段的同义词归并
- **质量检验**：缺失率统计、抽样准确率验证
- **研究设计辅助**：文献方法论蒸馏 + 数据EDA → 因素生成

## 使用方式

直接在对话中描述你的研究意图即可，例如：
- "我想研究劳动争议中法院支持劳动者的影响因素"
- "帮我处理这批裁判文书"
- "帮我分析这些论文的研究方法"

Skill 会主动引导你完成研究设计并启动处理流水线。**不需要编辑任何配置文件。**
